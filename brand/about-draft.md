# 旺旺の物語

## 65 歳から、はじまる物語

中国・山東省で生まれました。
長く、教育と行政の仕事に携わってきました。

2020 年、定年。
それまで前だけを向いて走ってきた日々が、ふと止まりました。

ちょうど世界が静かになった時期でした。
家にいる時間が増え、自然と「これからの自分」について考えるようになりました。

---

## 一つの旅が、変えてくれたこと

2024 年の夏、初めて日本を訪れました。

町を歩きながら、何度も足を止めました。
60 歳を、70 歳を、80 歳を過ぎても、自分の役割を持ち、自分の場所で一生懸命に手を動かしている方たち。
その姿が、忘れられませんでした。

──いくつになっても、自分の力の届く範囲のことを、丁寧にやれば、それは素敵なことなのだ。

帰国してからも、その光景が胸に残っていました。

---

## 中国と日本のあいだに、立ちたい

中国の暮らしの中で、何百年も前から受け継がれてきた、季節と身体を慈しむ道具と知恵。
それを、日本の丁寧な毎日に、そっと届けることができたら。

夫が応援してくれました。
長年の貯えを、思いきって日本での新しい一歩のために使うことにしました。

2024 年末、大阪に拠点を構え、太旺旺株式会社を立ち上げました。

---

## わたしたちが、大切にしていること

急がないこと。
飾らないこと。
ただ、ひとつひとつを丁寧に。

中国と日本のあいだに静かに立ち、文化を、暮らしを、橋渡しすること。
65 歳から始めた小さな会社ですが、焦らず、続けていきたいと思っています。

---

太旺旺株式会社　代表
楊　杰敏（ヤン　ジェミン）

## 編集ノート（内部用・公開しない）

### 削除した内容と理由

| 削除内容 | 理由 |
|---|---|
| 生年月日 1961-01-07 | 個人情報保護 |
| 「共青団済寧市委」 | 政治的に敏感、日本受众语境で不要な連想 |
| 「局長」具体的役職 | 個人情報＋政治的連想 |
| 中国・山東省**済寧市**（細かい地名） | 個人特定の余地を残さない |
| 「子どもが上海で生活」 | 家族プライバシー |
| 退職金・年金からの出資（具体金額） | 财务プライバシー |
| 在留資格・visa 申請の言及 | 文脈不適切（読み手は申請審査官ではない） |
| 1976–1984 / 1994.12 等の具体年月 | 細かすぎ、ブランド物語の流れを妨げる |
| 「事業が安定に連れ更に多くの素晴らしい商品と新しいビジネスを開拓し」 | ビジネス語気強すぎ、voice 不一致 |

### 保留した内容と理由

| 保留内容 | 理由 |
|---|---|
| 中国・山東省出身 | 出身地の地方性は信頼感を生む（市まで言わない） |
| 教育と行政の仕事 | 経歴の信頼性、政治色を消した generalized 版 |
| 2020 年定年 | ストーリー上の起点として必須 |
| COVID 後の自己反省 | 「ちょうど世界が静かになった時期」と婉曲表現 |
| 2024 年日本旅行 | 物語の核心 |
| シニア労働者を見た経験 | ブランド哲学の根拠、最重要シーン |
| 「いくつになっても…」の一文 | ブランド信条として最重要 |
| 夫の応援 | 信頼信号（過度ではない範囲で） |
| 大阪拠点 | 事業の事実 |
| CEO 本名 | 代表者として明示する責任 |

### Voice 改写の比較例

| 申請書原文 | 改写版 |
|---|---|
| 「私は、中国山東省の出身で、現在は上海に住んでおります」 | 「中国・山東省で生まれました」 |
| 「1976年から1984年までは教育に関わる仕事に従事いたしました。その後1994年12月から共青団済寧市委に転職し、以来公務員として2020年2月に定年まで勤めておりました」 | 「長く、教育と行政の仕事に携わってきました。2020 年、定年」 |
| 「日本の町中で多くのシニア労働者が一生懸命頑張っている姿を見かけたことでした。いくつになっても自分の力範囲内のことを一生懸命やれば素敵だと思いました」 | 「60 歳を、70 歳を、80 歳を過ぎても、自分の役割を持ち、自分の場所で一生懸命に手を動かしている方たち。その姿が、忘れられませんでした。──いくつになっても、自分の力の届く範囲のことを、丁寧にやれば、それは素敵なことなのだ」 |
| 「微力ですが、日本で一生懸命に頑張っていき、日中の懸け橋になるように努めてまいりたいです」 | 「中国と日本のあいだに静かに立ち、文化を、暮らしを、橋渡しすること」 |

### 構造の変更

- 申請書 = 時系列の事実陳述（学歴→職歴→動機→申請）
- 公開版 = 3 つの章立て（過去・転機・現在）+ 結びの信条
- 段落間に空行を挟み、呼吸感を作る
