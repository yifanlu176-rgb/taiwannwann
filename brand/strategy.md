## 1. Brand Foundation

### 1.1 Positioning Statement

> **太旺旺は、中国の暮らしの知恵と、日本の丁寧な毎日を結ぶ、生活文化のブランドです。**
> **65 歳でこの一歩を踏み出した一人の女性の物語から始まり、年齢や国境にとらわれず、自分のペースで豊かに生きることを、そっと支えています。**

> （中文意译・内部参考用）
> 太旺旺是连接「中国生活智慧」与「日本细致日常」的生活文化品牌。它始于一位女性 65 岁迈出的这一步，温柔地支持人们不被年龄与国境束缚，按自己的节奏丰盛地生活。

### 1.2 Mission

> 中国の暮らしの中で培われてきた、季節と身体を慈しむ道具と知恵を、日本の毎日にそっと届ける。

### 1.3 Vision（3 年）

> 「中国と日本のあいだ」に静かに立ち、商品・物語・体験を通じて、暮らしを慈しむ文化を伝える小さなブランドであり続けること。

**「小さなブランドであり続けること」は意図的選択**。量産・大規模化を追わない。「大きくならない」ことを誇りにする。

### 1.4 Core Values

| 日本語 | 解説 |
|---|---|
| **慈しむ**（Itsukushimu） | 大切にする、丁寧に扱う ── 物への態度 |
| **馴染む**（Najimu） | 環境にすっと溶け込む、押し付けない ── 場への態度 |
| **続ける**（Tsuzukeru） | 一過性ではなく、長く静かに ── 時間への態度 |

## 3. Tone of Voice

### 3.1 Voice Matrix

| 是 | 不是 |
|---|---|
| 温かい（warm） | 馴れ馴れしい（overly familiar） |
| 静か（quiet） | 退屈・地味（boring） |
| 真摯（sincere） | 重い・説教的（preachy） |
| 丁寧（careful） | 古臭い（old-fashioned） |
| 知性的（intelligent） | 知識を見せびらかす（showing off） |
| 物語性のある（narrative） | おしゃべり（chatty） |
| ゆったりした（slow） | 怠惰（lazy） |

### 3.2 文体ガイドライン

- **基底（90%）**：現代日本語随筆語気。参考：『暮しの手帖』『&Premium』、角田光代の散文
- **高光（10%）**：tagline / philosophy / hero 段落では文語的余韻
- **人称**：通常は第三人称、時々「私たち」、CEO 本人の声では「わたし」
- **一文の長さ**：短め。30 文字を意識上限に
- **読点**：少なめ。リズム優先
- **漢字とひらがなのバランス**：ひらがな多め（やわらかさ）

### 3.3 Voice Sample（既存 → 改写）

| 既存（NG / グレー） | 改写（OK） |
|---|---|
| 首・肩のコリをやさしくほぐし、質の高い睡眠をサポートします | 天然素材のビーズを使った、伝統的な意匠の枕。お部屋に置いておくだけで、中国の暮らしの一端に触れることができます |
| 経絡治療儀 | 経絡をモチーフにした、中国伝統の意匠の道具 |
| 健康器具の輸入販売 | 中国の伝統雑貨と暮らしの道具を、日本へ届けています |
| 中国と日本をつなぐ健康器具の輸入販売、そして飲食事業の展開により、皆さまの食・健康・くらしを丁寧にサポートいたします | 中国の暮らしの中で受け継がれてきた、季節と身体を慈しむ道具と知恵を、日本の毎日にそっと届けています。これから、食の事業も、ゆっくりと始めていきます |

## 4. Do / Don't Word List

### 4.1 薬機法红线（絶対 NG）

| カテゴリ | NG 例 |
|---|---|
| 効能効果の暗示 | 効果 / 効能 / 効く / 改善 / 緩和 / 軽減 / 治る / 治療 / 治癒 / 予防 / 防ぐ |
| 身体機能サポート系 | ほぐす（部位 + ほぐす）／ サポート（生理機能を）／ 健康になる ／ 健康に良い |
| 症状言及 | 痛み ／ 凝り ／ 不眠 ／ 疲れ（身体の） |
| 医療連想 | 治療儀 ／ 療法 ／ 施術 ／ 患部 |

### 4.2 政治・社会的红线

| 項目 | 取り扱い |
|---|---|
| 「共青団」「共産党」「中共」 | **絶対 NG**。CEO 経歴は「中国地方政府で 30 年余り、教育・行政分野の公職」と表記 |
| 「中国製」「Made in China」 | 強調しない。事実として記載する程度（規格・原産国欄）に留める |
| 中日関係への言及 | しない。政治的に中立 |
| 「シニア」「高齢」 | 強調しない。65 歳の物語は語るが、シニア限定の商品では **ない** |
| 民族感情 | 触れない |

### 4.3 推奨される代替表現

| NG | OK |
|---|---|
| 肩こりをほぐす | リラクゼーションのひととき |
| 健康器具 | 伝統雑貨 ／ ライフスタイルアイテム ／ 暮らしの道具 |
| 効果を実感 | 暮らしに馴染む |
| サポート | 寄り添う ／ 添える |
| 改善 | 慈しむ |
| 治療儀 | 中国伝統の意匠 ／ 経絡をモチーフにした道具 |
| 健康になる | 自分のペースで生きる |
| 中国製造 | 中国伝統 ／ 中国の暮らしの知恵 |
| シニア ／ 高齢者 | 二章目 ／ これからの日々 ／ どんな世代にも |

### 4.4 ブランド独自フレーズ

| フレーズ | 用途 |
|---|---|
| 旺旺と、豊かに生きる | Tagline（既存・保留） |
| 65 歳から、はじまる物語 | About 導入 ／ Hero |

## 5. Visual Direction

### 5.1 Color System

```
Base（既存・保留）:
  --cream    #f7f2ea  - 主背景
  --paper    #faf7f2  - 副背景
  --warm     #c8956c  - 主アクセント（暖色）
  --moss     #7a8c6e  - 副アクセント（緑）

Add（新規追加）:
  --sumi     #1a1812  - 墨色：文字、信念感のある重み
  --kintya   #b8915c  - 金茶：見出し / hover / 強調点
  --shu      #a8412a  - 朱：年に数回的な強調 only（< 1% 面積）
```

**哲学**：日式克制 × 中式底蕴。朱色（#a8412a）面積を全画面 < 1% に抑える。**致敬中国伝統色但不浮夸**。

### 5.2 Typography

| 用途 | フォント |
|---|---|
| 日文 body / heading | Noto Serif JP（保留） |
| 中文（繁体） | **Noto Serif TC**（既存 Noto Serif SC から差し替え推奨。受众繁体に合わせる） |
| Latin / 数字 | Cormorant Garamond（保留） |
| Mono / コード | system mono（必要時のみ） |

スケール（Day 2 `tokens.md` にて具体定義）：display / h1 / h2 / h3 / body / caption。

### 5.3 Imagery Direction

| カテゴリ | 方向性 |
|---|---|
| 光 | 午後の自然光、室内の影、北窓の柔らかさ |
| 季節感 | 春＝桜・新緑 ／ 夏＝青葉・水滴 ／ 秋＝銀杏・暖色 ／ 冬＝白・木の質感 |
| 構図 | 余白 60–70% OK、商品は控えめに |
| 色 | 暖色寄り、彩度低め |
| 人物 | 手元・後ろ姿・遠景中心。顔写真は控えめ（CEO 写真は About に 1 枚程度） |
| 質感 | 木・紙・布・陶器・籐 |
| NG | 白底電商商品写真をそのまま掲載（商品グリッド以外）／ ストックフォト感丸出しの笑顔 ／ 過度なフィルター |

### 5.4 Motion Principles

| カテゴリ | 方向性 |
|---|---|
| ベース | Subtle fade-up（既存 IntersectionObserver で OK） |
| Hover | Color shift ／ scale 1.02 ／ 下線描画 程度 |
| Key visual | 線の描き出し（hero に 1 箇所のみ） |
| Page transition | Soft fade（300ms） |
| NG | 大きな motion ／ parallax ／ 進站動画 ／ 自動再生動画 |

## 6. Brand Personality

### Archetype

**The Sage（70%）+ The Caregiver（30%）**

- The Sage — 知恵 ／ 静けさ ／ 真摯 ／ 経験 ／ 観察
- The Caregiver — 温かさ ／ 寄り添い ／ 慈しみ ／ 共感

### Personification

> 「人生の二章目を、丁寧に歩み始めた、ひとりの女性」

- 65 歳前後
- 静かで聡明
- 急がない
- 国境と年齢を越えて、自分のペースで生きている
- 直接 CEO 楊杰敏ではない（personal brand 化を避ける）
- 「彼女のような人」を体現する集合的なイメージ

### Personality Test

「もしこのブランドが部屋に来たら、何をする？」
- 静かにお茶を淹れる
- 窓辺で本を読んでいる
- 急かさない
- 黙って一杯の中国茶を出してくれる

「もしこのブランドが SNS で何かを発信したら？」
- 一日一枚の写真と一言
- セールではなく、季節の節目
- リプライは丁寧に、でも頻繁ではない

**About**
- visa 信件素材から、隐私と政治的標識を removed した公開版を使用
- 草案は `brand/about-draft.md` 参照
- CEO の名前と「代表」「楊　杰敏（ヤン　ジェミン）」は記載 OK
- 写真は控えめに 1 枚（後ろ姿、手元、または横顔の遠景）

**Products**
- 全 6 商品を「文化雑貨 / lifestyle」として再定位
- 各商品に「なぜ選んだか」の小さな物語（200 字程度）を追加
- 効能効果の暗示を一切しない
- 「経絡治療儀」→「中国伝統の意匠の道具」等に改名

**Channels（重要・修正必須）**

ユーザー確認済み：**商品はまだ Yahoo!ショッピング・楽天市場には未上架**。

現在の文案「Yahoo!ショッピング・楽天市場・SNS にてお取扱中」は景品表示法上の不実表示。

修正方向（いずれか）:
- 「近日、Yahoo!ショッピング・楽天市場でもお求めいただけるよう、準備中です」
- 「ご購入・お取扱いについては、お問い合わせください」
- SNS（LINE / WeChat / Instagram / Facebook）は事実 OK なら明記してよい

**Restaurant**
- 「Coming Soon」保留
- CEO の一言を追加：「料理にも、暮らしの知恵が宿ります」程度

**FAQ（Week 3 Day 5 作成）**
- 「これは医療機器ですか？」→「いいえ、すべて雑貨・ライフスタイル商品です」
- 「効果は保証されますか？」→「効果効能を謳う商品ではございません」
- 「中国製ですか？」→「はい、中国の職人さんや工房から仕入れています」
- 「いつ実店舗ができますか？」→「現在準備中です」
