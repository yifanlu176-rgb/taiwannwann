# taiwannwann-website

当前仓库是从 `https://github.com/yifanlu176-rgb/taiwannwann` 拉取下来的静态单页站点。

## 当前状态

- 入口页：`index.html`
- 页面内容：公司介绍、业务、商品、饮食事业、关于我们、联系表单
- 技术形态：纯 HTML + 外链 CSS / JS，暂未引入构建工具

## 建议目录

```text
.
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   └── img/
│       └── products/
└── docs/
    └── structure.md
```

## 页面区块

- `hero`：首屏主视觉
- `philosophy`：企业理念
- `business`：业务介绍
- `products`：商品展示
- `restaurant`：饮食事业
- `about`：公司概要
- `contact`：联系表单
- 联系表单已接入 Formspree
- `footer`：页脚信息
