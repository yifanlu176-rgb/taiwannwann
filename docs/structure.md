# 目录结构说明

仓库当前以 `index.html` 为主入口，适合继续按静态站方式推进。

## 现有代码归类

- 页面结构：`index.html`
- 视觉样式：`assets/css/styles.css`
- 页面脚本：`assets/js/main.js`
- 商品模块：`assets/js/products.js`
- 站点内容模块：`assets/js/site-content.js`
- 联系表单：Formspree `https://formspree.io/f/maqkwwqv`
- 产品图片：`assets/img/products/`

## 预留目录

- `assets/css/`：后续拆出的样式文件
- `assets/js/`：后续拆出的交互脚本
- `assets/img/products/`：产品图片
- `assets/img/`：其他图片、Logo、示意图

## 推荐任务切分

1. 页面结构整理
2. 样式模块化
3. JS 交互模块化
4. 内容更新和素材补齐
